    Access Facebook on Terminal


    1. Check Notifications
    2. Check who is online
    3. Post a status
    4. And more...(Coming soon)

    Requirements

    • Python 2.7.3
      $ apt-get install python
    • Python PIP
      $ apt-get install python-pip
    • PyCurl
      $ pip install pycurl

    Installation

    • Fast forward 
      $ pip install fbterminal
    • Installing from source 
      $ git clone git@github.com:iAkashz/fbterminal.git
      $ python setup.py install
    • Go to Facebook Developers and create a new app
      Copy the APP ID and APP SECRET to file ~/.fbterminal

    Running

    $ fbterminal -p 'Hello World'
