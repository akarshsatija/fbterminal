Access Facebook on Terminal


  1. Check Notifications
  2. Check who is online
  3. Post a status
  4. And more...(Coming soon)

Requirements

  • Python 2.7.3
    $ apt-get install python
  • Python PIP
    $ apt-get install python-pip
  • PyCurl
    $ pip install pycurl

Installation

  • Fast forward 
    $ pip install fbterminal
  • Installing from source 
    $ git clone git@github.com:decached/fbterminal.git
    $ python setup.py install
  • Go to Facebook Developers and create a new app
    1. Add `App Name` of your choice
    2. Deselect - `Web Hosting` (You won't be needing that)
    3. Enter the Captcha
    4. In `App Domains` put `localhost`
    5. Select `Website with Facebook Login`
    6. In `Site URL` put `http://localhost:7777/`
    7. Save Changes
    Copy the APP ID and APP SECRET to file ~/.fbterminal

Running

$ fbterminal -p 'Hello World'
